Que es la WWW

La World Wide Web (Web), es una red de recursos de información.

  1. Un esquema uniforme de nombres para localizar recursos en la Web (p.ej., URIs).
  2. Protocolos, para acceder a recursos con nombre en la Web (p.ej., HTTP).
  3. Hipertexto, para navegar fácilmente entre recursos (p.ej., HTML).

CSS: Media Queries

@media 
                
  @media print {
    body { font-size: 10pt }
  }
  @media screen {
    body { font-size: 13px }
  }
  @media screen, print {
    body { line-height: 1.2 }
  }
  @media only screen
    and (min-width: 320px)
    and (max-device-width: 480px)  {
      body { line-height: 1.4 }
  }

Mixin funciones

Las variables que definimos en un mixin se pueden usar en la regla que lo invoca. 

                
.average(@x, @y) {
  @average: ((@x + @y) / 2);
}

div {
  .average(16px, 50px); // "call" the mixin
  padding: @average;    // use its "return" value
}
